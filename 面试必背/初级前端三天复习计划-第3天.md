## 📖 详细学习路径

### 第一部分：HTTP网络（2.5小时）

#### ✅ 必学知识点

1. **HTTP vs HTTPS** → [HTTP面试题精简版.md](./HTTP面试题精简版.md#1-http-vs-https-)
   - 区别：端口、安全性、证书、速度
   - HTTPS加密原理：对称加密 + 非对称加密
   - SSL/TLS握手过程
   - CA证书的作用

2. **GET vs POST** → [HTTP面试题精简版.md](./HTTP面试题精简版.md#2-get-vs-post-)
   - 参数位置：URL vs 请求体
   - 数据大小限制
   - 幂等性：GET幂等、POST非幂等
   - 缓存：GET可缓存、POST不可缓存
   - RESTful API中的应用

3. **HTTP状态码** → [HTTP面试题精简版.md](./HTTP面试题精简版.md#4-http-状态码-)
   - 2xx 成功：200、201、204
   - 3xx 重定向：301、302、304
   - 4xx 客户端错误：400、401、403、404
   - 5xx 服务器错误：500、502、503

4. **HTTP缓存** → [协商缓存和跨域面试题.md](./协商缓存和跨域面试题.md#一强缓存)
   - 强缓存：Cache-Control、Expires
   - 协商缓存：ETag/If-None-Match、Last-Modified/If-Modified-Since
   - 缓存策略最佳实践
   - 浏览器缓存机制

5. **跨域解决方案** → [协商缓存和跨域面试题.md](./协商缓存和跨域面试题.md#三跨域问题)
   - 同源策略：协议、域名、端口
   - CORS：预检请求、响应头设置
   - JSONP：原理和局限性
   - 代理服务器：开发环境和生产环境

### 第二部分：异步编程（2.5小时）

#### ✅ 必学知识点

1. **Promise** → [JS高频面试题汇总.md](./JS高频面试题汇总.md#15-promise)
   - 三种状态：pending、fulfilled、rejected
   - 基本使用：then、catch、finally
   - 解决回调地狱问题
   - Promise.all、Promise.race、Promise.allSettled
   - 手写Promise.all

2. **async/await** → [JS高频面试题汇总.md](./JS高频面试题汇总.md#17-asyncawait)
   - 语法糖本质：基于Promise
   - async函数返回Promise
   - await等待机制
   - 错误处理：try-catch
   - 串行 vs 并行执行
   - 性能优化：避免不必要的await

3. **事件循环** → [事件循环面试题详解.md](./事件循环面试题详解.md#一基础概念)
   - 宏任务 vs 微任务
   - 执行顺序：同步 → 微任务 → 宏任务
   - 常见宏任务：setTimeout、setInterval、I/O
   - 常见微任务：Promise.then、MutationObserver
   - 经典题目分析

### 第三部分：AJAX（1小时）

#### ✅ 必学知识点

1. **XMLHttpRequest** → [AJAX经典面试题.md](./AJAX经典面试题.md#二xmlhttprequest)
   - 五种状态：0-4的含义
   - 基本使用：open、send、onreadystatechange
   - GET和POST请求示例

2. **Fetch API** → [AJAX经典面试题.md](./AJAX经典面试题.md#三fetch-api)
