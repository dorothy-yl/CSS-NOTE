   - 前后端联调效率提升，接口错误在开发阶段就能发现

# 性能优化类问题

**Q: 你在项目中做过哪些性能优化？**

**回答框架（多个案例）：**

**1. Next.js SSR优化（Image SaaS项目）**
- **问题**：首屏加载慢，大量API请求在客户端执行
- **方案**：利用Server Components在服务端直接通过tRPC Server Caller调用API
- **效果**：首屏加载时间减少40%，减少了客户端JavaScript体积

**2. 小程序分包优化（慕尚花坊项目）**
- **问题**：主包体积超限，首次加载时间长
- **方案**：采用主包+分包架构，将商品详情、订单、个人中心等模块分包加载
- **效果**：首屏加载时间从5s降低到2s以内

**3. 列表渲染优化（壁纸项目）**
- **问题**：图片列表滚动卡顿
- **方案**：实现虚拟列表+图片懒加载+分页加载
- **效果**：支持流畅浏览上千张图片

**4. 请求优化（Image SaaS项目）**
- **问题**：多个API请求导致页面加载慢
- **方案**：配置tRPC HTTP Batch Link实现请求批处理
- **效果**：相关请求合并为一次HTTP调用，减少网络往返

**5. 通用优化手段**
- 防抖节流：搜索框输入、滚动事件
- 图片优化：压缩、webp格式、CDN加速
- 代码分割：按需加载、Tree Shaking
- 缓存策略：浏览器缓存、HTTP缓存、本地存储

## 应急话术（救场用）

**1. 被问到不会的技术问题：**
> "这个技术点我目前还没有深入实践过，但我的理解是...[说出你知道的相关内容]...我会在工作之前系统地学习一下，请问您能推荐一些学习资源吗？"

**2. 项目被质疑不够复杂：**
> "您说得对，这个项目确实不算特别复杂，但对我来说它帮助我建立了完整的开发流程认知，从需求到设计、开发、测试、部署的全链路。我也意识到需要接触更有挑战的项目，这也是我来应聘贵公司的原因。"

**3. 简历有间隙被问到（毕业后2-3个月学习期）：**
> "毕业后我没有急着找工作，而是花了2-3个月时间系统地学习前端进阶技术，包括React生态、Next.js全栈开发、Uni-app跨平台开发等。我认为这段时间的投入是值得的，它让我在后续的项目开发中能够快速上手，独立完成复杂功能。"

**4. 被问为什么换工作频繁（外包项目切换）：**
> "外包项目的特点是周期短、项目切换快，这让我能在短时间内接触多个领域的项目。但现在我更希望在一个产品上做深入的技术积累，而不是浅尝辄止。"

**5. 年龄/学历被质疑：**
> "我理解您的顾虑。虽然我的学历不是计算机专业/不是本科，但我通过自学和实战项目证明了自己的学习能力和技术实力。我相信能力比学历更重要，也希望能有机会证明自己。"
